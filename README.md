Hello, this is my first learning project in python! This is the Quest of the Eternal Caverns.
(Game is still in beta, expect bugs and messy code)


                           --]| FEATURES |[-- 

- Immersive Storytelling and fun choice mechanic in general gameplay
- Unlimited replayability with over 100 combinations of custom race + classes
- Great strategic gameplay especially in combat and gold economy.
                          --]| Gameplay |[--
- 10 unique player classes (Warrior, Mage, Roge, etc...)
- 10 unique player race (Human, Kithling, Lunari, etc...)
